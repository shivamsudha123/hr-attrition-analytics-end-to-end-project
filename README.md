# hr-attrition-analytics-end-to-end-project
## Description

1. HR Attrition Analytics End To End Project  is a Data Analyst project based on SQL and Power BI.

2. The project focuses on identifying factors affecting employee attrition.

3. Employee data such as age, department, and job satisfaction was analyzed.

4. Work-life balance, salary hike, overtime, and business travel were also examined.

5. Interactive dashboards and visualizations were created in Power BI.

6. Business insights were generated to understand employee turnover trends.

7. Recommendations were provided to improve employee retention and workforce management.
